Web 1.0

Explore the early technologies, languages, and tools that laid the foundation for the World Wide Web, starting with the creation of HTML, HTTP, and early scripting languages.

3.1 HTML / HTTP Protocol

Who

Tim Berners-Lee, a British computer scientist, invented the World Wide Web. He developed core technologies like HTML and HTTP.

What

HTML (Hypertext Markup Language) structures web pages, while HTTP (Hypertext Transfer Protocol) facilitates communication between web browsers and servers.

Why

These technologies were created to make sharing information on the internet easier and more accessible.

When

Berners-Lee developed the first versions of HTML and HTTP between 1989 and 1990. The first website went live in 1991.

How

HTML structures web content, while HTTP defines how data is exchanged. This interaction formed the basis of the first static websites.

3.2 JavaScript

Who

Brendan Eich, a programmer at Netscape, created JavaScript in 1995 to make web pages interactive.

What

JavaScript brought interactivity to static Web 1.0 pages, enabling features like form validation and pop-ups.

Why

It allowed developers to add interactivity without requiring a full page reload.

How

JavaScript manipulates HTML elements within the browser, enhancing user interaction. Learn more at Mozilla Developer Network.
